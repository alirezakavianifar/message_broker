Certificate Management Information
Important Notes:
  • Generated certificates use mutual TLS for secure client authentication
  • Certificates are automatically validated against the Certificate Revocation List (CRL)
  • Certificate files are stored in main_server/certs/ directory
  • Revoked certificates are logged in the audit trail
  • Use the admin CLI for advanced certificate operations: python admin_cli.py cert list
Certificate Distribution:

After generating a certificate, securely distribute the following files to the client:

  1. client_id.crt - Client certificate
  2. client_id.key - Private key
  3. ca.crt - CA certificate
